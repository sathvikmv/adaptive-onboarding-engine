# 🚀 ElevateAI V3 — Adaptive Onboarding Engine

> AI-powered system that analyzes resumes, matches job roles, and generates personalized learning pathways.

---

## 🧠 Overview

ElevateAI is an intelligent onboarding engine designed to reduce time-to-competency by:

- 📄 Parsing resumes
- 🎯 Matching job descriptions
- 🧩 Generating adaptive learning paths
- 🌐 Visualizing knowledge graphs

---

## ⚙️ Tech Stack

### 🧩 Backend
- FastAPI
- Python
- NetworkX (graph processing)
- PyPDF (resume parsing)

### 🎨 Frontend
- React (Vite)
- TypeScript
- Tailwind CSS

### 🐳 DevOps
- Docker
- Docker Compose

---

## 🚀 Features

- ✅ Resume Upload & Parsing  
- ✅ Skill Extraction  
- ✅ AI-based Learning Pathway Generation  
- ✅ Graph Visualization  
- ✅ Full Dockerized Setup
