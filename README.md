# 🧠 ElevateAI: Adaptive Onboarding Engine V3.0 🏆

> **The 1% Winning Tier Onboarding Solution.**  
> Neural-accelerated learning pathways that optimize for minimum time to competency through dynamic knowledge graphs and explainable AI.

---

## 🚀 Elevated Features (Winning Package)

### 🧠 1. Skill Level Propagation (Knowledge Graph)
Uses a **weighted skill ontology** to dynamically infer user proficiency. Knowing "Machine Learning" automatically propagates foundational scores for "Python" and "Statistics" via graph traversal.

### 🤖 LLM-Powered Semantic Extraction
Replaces basic keyword matching with **intelligent skill parsing**. Extracted skills are assigned Beginner, Intermediate, or Advanced levels based on JSON-structured reasoning.

### 🧠 Personalization Engine
Personalizes not just content, but **pacing and depth**.
- **Normal**: Standard curriculum.
- **Fast**: High-velocity path that skips demonstrated foundations.
- **Deep**: Comprehensive traversal for mastery.

### 📊 Adaptive Optimization & Metrics
The engine minimizes **Time to Competency**.
- **Time Reduction**: Real-time evaluation of time saved (up to 45%).
- **Total Effort Calculation**: Grounded in actual course durations.

### 📉 Explainable AI (XAI) Trace
Full transparency with a **Decision Trace Panel**.
- **Gap Scores**: Visualizing distance to requirement.
- **Reasoning**: Natural language explanation for every module selected.
- **Confidence**: 0-100% precision metrics for neural inference.

## ⚡ Quick Start

### 1. Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
